# VPS Auto-WARP & Optimization Configurator (unlock-ai)

Скрипт для локального запуска на компьютере (macOS, Linux или Windows через Git Bash), который автоматически настраивает обход ограничений для нейросетей (**Gemini, ChatGPT, Claude**) на вашем удаленном Linux VPS-сервере через Cloudflare WARP.

Скрипт полностью совместим с **AmneziaVPN** и **3x-ui (Xray)**, не ломает существующие VPN-соединения и порты, а также выполняет оптимизацию сети (BBR + TCP) и повышает скрытность сервера (скрытие от пинга ботами).

---

## Что делает скрипт:
1. **Подключается по SSH** к вашему серверу, используя введенные вами IP, порт и имя пользователя (пароль запрашивается безопасно самой утилитой SSH и не сохраняется скриптом).
2. **Устанавливает `sing-box`** на стороне сервера как фоновую службу.
3. **Регистрирует новое устройство в Cloudflare WARP** и настраивает WireGuard-туннель.
4. **Конфигурирует правила маршрутизации:** трафик к Gemini, ChatGPT и Claude идет через WARP (меняется IP-адрес для этих сайтов), весь остальной трафик идет напрямую через ваш родной IP сервера.
5. **Блокирует протокол QUIC (UDP 443)** для заблокированных сайтов, чтобы форсировать их работу по TCP (это необходимо для работы прозрачного прокси).
6. **Оптимизирует параметры сети (BBR + TCP)** в зависимости от объема оперативной памяти (RAM) сервера, чтобы избежать перегрузки памяти (OOM-killer).
7. **Отключает ответы на пинг (ICMP Echo)** для защиты сервера от сканирования ботами.

---

## Требования:
* **Локально:** macOS, Linux или Windows с установленным Git Bash (входит в стандартную установку Git для Windows) или WSL.
* **На сервере:** Установленная ОС Ubuntu (20.04+) или Debian (10+).

---

## Как запустить скрипт локально:

1. Откройте терминал на вашем устройстве (macOS Terminal, Linux Terminal, или Git Bash на Windows).
2. Скачайте и запустите скрипт:
   ```bash
   chmod +x unlock-ai.sh
   ./unlock-ai.sh
   ```
3. Следуйте подсказкам на экране.

---

## Как разместить скрипт на GitHub и запускать одной строкой:

Если вы хотите, чтобы ваш друг мог запустить скрипт одной командой без предварительного скачивания файлов, вы можете выложить его на GitHub (или GitHub Gist):

### Шаг 1: Создание репозитория
1. Зайдите на свой аккаунт GitHub.
2. Создайте новый публичный репозиторий (например, с именем `warp-setup`).
3. Загрузите файл `unlock-ai.sh` в этот репозиторий.

### Шаг 2: Получение прямой ссылки (Raw Link)
1. Откройте загруженный файл `unlock-ai.sh` на GitHub.
2. Нажмите кнопку **"Raw"** в правом верхнем углу просмотра кода.
3. Скопируйте адрес из адресной строки браузера. Он будет выглядеть примерно так:
   `https://raw.githubusercontent.com/ВАШ_НИКНЕЙМ/warp-setup/main/unlock-ai.sh`

### Шаг 3: Запуск одной командой
Теперь ваш друг может просто открыть терминал на своем компьютере и выполнить:
```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/ВАШ_НИКНЕЙМ/warp-setup/main/unlock-ai.sh)"
```
Это скачает скрипт в оперативную память и сразу запустит его с красивым приветствием!
